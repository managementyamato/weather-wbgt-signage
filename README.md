# 天気・暑さ指数モニター

GPSで現在地を取得し、天気情報とWBGT（暑さ指数）を表示するダッシュボード。

## ファイル構成

```
weather-dashboard/
├── index.html    # メインのダッシュボード
└── README.md     # このファイル
```

## 機能

- **天気情報**（気象庁API）
  - 天気アイコン・説明
  - 気温（午前: 最低気温 / 午後: 最高気温）
  - 降水確率

- **暑さ指数（WBGT）**
  - 夏季（4月下旬〜10月下旬）のみ表示
  - データがない場合は非表示

## データソース

| データ | ソース |
|--------|--------|
| 天気 | 気象庁API (`www.jma.go.jp/bosai/forecast/`) |
| WBGT | 会社APIプロキシ経由 |

## 使い方

1. `index.html`をWebサーバーにホスティング
2. ブラウザでアクセス
3. 位置情報の許可を求められたら許可

※ローカルファイル（file://）では位置情報APIが動作しません

## 更新間隔

5分ごとに自動更新

## 対応地域

全国47都道府県の主要都市
